# 塔罗占卜 Web 应用实现计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 构建一个基于 React + TypeScript + Vite 的塔罗占卜 Web 应用，支持单张和三张牌阵，含翻牌动画和中文解读。

**Architecture:** 纯前端静态应用，无后端。数据硬编码在 `src/data/cards.ts`，抽牌逻辑封装在自定义 Hook，页面分首页和占卜页两个路由。

**Tech Stack:** React 18, TypeScript, Vite, CSS Modules, React Router v6
